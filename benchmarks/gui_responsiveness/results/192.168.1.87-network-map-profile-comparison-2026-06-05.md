# GUI Responsiveness: Network Map State Profile

Target: `http://192.168.1.87:8877/`

Compared runs:

- Before: `192.168.1.87-after-status-console-profiles-full-2026-06-05.json`
- After: `192.168.1.87-after-network-map-profile-full-2026-06-05.json`

## Change

Added `profile=network-map` for the network map subview.

The profile keeps:

- Slim node rows
- Slim network edges for real and estimated map links
- Tiny recent-packet activity records for map node flashing

The profile removes:

- Recent chat rows
- Packet bodies and decoded payload blobs
- Port count summaries
- Node packet trend payloads

Graph, routes, and top-node network subviews continue to use `profile=network`.
